More new features this weekend by popular demand:


- Ability to set Page Numbering footer size
- Ability to set Page Numbering colour to black, red or blue
- Font compatibility: Times and Arial style alternatives
- Usability tweaks eg to how options are presented


For the fonts, BunTool only uses free and open fonts so neither Times New Roman nor Arial are appropriate. I am told that some courts require use of these fonts, which is surprising but probably borne of ignorance not an intention to tie court users down. I think Liberation Sans and Charis SIL make fantastic alternatives which pass for Times and Arial styles without being proprietary. If anything I think they look better. I tried the TeX Gyre family, but it's chiefly in OTF format and jsPDF wants TTF fonts.
