# GSTAT Index Generation Tool

**Automatically create GSTAT ready bundles in seconds.**

The tool is used to take in PDFs and produce perfect GSTAT website bundles. It processes PDF files and generates merged outputs with:

- Automatic index/table of contents
- Hyperlinked page numbers
- PDF bookmarks/outlines
- Consistent page numbering
- Section breaks
- Common GSTAT bundle in compliance with Question 20 of the GSTAT appeal Checklist
- Ability to download "sections" with intact page numbering.

Bundles comply with English court requirements but work well for any context requiring organised PDF compilation.

## Easiest Way to Use

It's hosted at [https://github.com/test2a/GSTAT-tool](https://github.com/test2a/GSTAT-tool) - no installation required, software runs locally on your machine.

## Privacy

The tool runs entirely in your browser. Your documents are **never uploaded** to any third party server by BunTool - all processing happens locally on your device.

## Fork 

This tool is a fork of [BunTool2](https://github.com/TrisSherliker/buntool2). It is a JavaScript library for creating professional PDF court bundles.
